# clash-to-v2rayn-sub-converter
Extracts config data for V2rayN from subscription links intended for happ/clash

New clients like Happ/Clash use HWID generated locally to get config data from a subscription link, V2rayN and some other cliens don't. Program generates a fake HWID to mimic happ/clash and make these requests to aquire config data then converts it for V2rayN.
